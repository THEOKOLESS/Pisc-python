Emphasis

Markdown
**bold text**
*italic text*
***bold and italic***
~~strikethrough~~
Result

bold text

italic text

bold and italic

strikethrough

Lists

Markdown — Unordered
- Item 1
- Item 2
  - Nested item
  - Nested item
- Item 3
Result
  • Item 1
  • Item 2
    • Nested item
    • Nested item
  • Item 3
Markdown — Ordered
1. First
2. Second
3. Third
Result
  1. First
  2. Second
  3. Third

Links

Markdown
[Link text](https://example.com)
[Link with title](https://example.com "My title")
Result

Link text

Link with title

Images

Markdown
![Alt text](image.png)
![Alt text](image.png "Optional title")
Result

<img> tag with alt text and optional title

Code

Markdown — Inline
`inline code`
Result

inline code

Markdown — Block
```python
def hello():
    print("Hello, world!")
```
Result
def hello():
    print("Hello, world!")
