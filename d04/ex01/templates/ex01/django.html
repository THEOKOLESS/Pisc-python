Ex01 : Django, framework web.

Qu'est-ce que Django ?

Django est un framework web open-source écrit en Python. Son objectif principal est de faciliter et d'accélérer le développement d'applications web en fournissant une structure solide, des composants réutilisables et de nombreuses fonctionnalités intégrées : gestion des URL, système de templates, ORM (Object-Relational Mapping), authentification, interface d'administration, etc.

Django suit le principe DRY (Don't Repeat Yourself) et adopte une architecture MVT (Model - View - Template), variante du célèbre patron MVC.

Historique

2003 — Django est créé par Adrian Holovaty et Simon Willison au sein du journal Lawrence Journal-World (Kansas, États-Unis), pour répondre aux besoins de publication web à grande cadence du journal.
2005 — Django est publié en open-source sous licence BSD. Le nom est un hommage au guitariste de jazz Django Reinhardt.
2008 — Création de la Django Software Foundation (DSF), organisation à but non lucratif chargée de soutenir le développement du framework.
2013 — Django 1.5 introduit le support expérimental de Python 3.
2017 — Django 2.0 abandonne Python 2 et devient exclusivement Python 3.
2019 — Django 3.0 introduit le support de l'ASGI, ouvrant la voie aux fonctionnalités asynchrones.
Aujourd'hui — Django est l'un des frameworks web Python les plus populaires, utilisé par Instagram, Pinterest, Mozilla ou encore Disqus.
